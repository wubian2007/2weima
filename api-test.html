🖼️ 图片地址更新API测试

📋 API接口信息

接口地址： POST /api/update-image

请求参数：

{ "imageUrl": "https://example.com/image.jpg" }

响应格式：

{ "success": true, "message": "图片地址更新成功", "data": { "imageUrl": "https://example.com/image.jpg", "updatedAt": "2025-08-12T08:30:00.000Z" } }

💡 使用示例

JavaScript调用示例：

fetch('/api/update-image', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ imageUrl: 'https://pay.qq.com/cgi-bin/account/get_qr_image.cgi?size=120&url=weixin%3A%2F%2Fwxpay%2Fbizpayurl%3Fpr%3Dexample' }) }) .then(response => response.json()) .then(data => { if (data.success) { console.log('更新成功:', data.data); } else { console.error('更新失败:', data.error); } });

cURL调用示例：

curl -X POST http://localhost:3000/api/update-image \ -H "Content-Type: application/json" \ -d '{"imageUrl": "https://example.com/image.jpg"}'
